## [2026-07-05] 任务：世界杯集锦链接更新

**经验**：咪咕视频的具体比赛链接（带数字ID的）无法通过搜索引擎找到，它们是动态生成的或需要登录才能获取。遇到这种情况应该直接告诉用户，而不是反复搜索浪费时间。

**场景**：需要获取特定视频平台的深层链接时

**置信度**：高

**验证次数**：1

## [2026-07-05] 任务：GitHub Pages部署

**经验**：GitHub Pages网站更新需要三个步骤：修改文件 → git commit → git push。只修改本地文件不会自动同步到线上。

**场景**：使用GitHub Pages部署静态网站时

**置信度**：高

**验证次数**：1

---

## [2026-07-05] 任务：命令行走代理

**经验**：Windows PowerShell命令行默认不走系统代理。需要手动设置环境变量 `$env:HTTP_PROXY` 和 `$env:HTTPS_PROXY`。 socks5 协议比 http 协议在Git推送中更稳定。

**场景**：在需要代理的网络环境下执行命令行工具时

**置信度**：高

**验证次数**：1
